# 📊 Sales Data Analysis & ETL Project

## 🔹 Overview

This project analyzes sales data using SQL and Power BI.
The goal is to clean, transform, and analyze the dataset to extract useful business insights.

---

## 🔹 Tools Used

* SQL (Data Cleaning & Analysis)
* Power BI (Data Visualization)
* Excel / CSV (Data Source)

---

## 🔹 ETL Process

* Extract: Imported data from CSV file
* Transform: Cleaned data using SQL (handling NULLs, creating new columns)
* Load: Stored cleaned data in database

---

## 🔹 Key Analysis

* Top Customers by Sales
* Best Selling Products
* Monthly Sales Trends
* Sales by Category
* Profit Analysis

---

## 🔹 Key Insights

* Identified top-performing customers
* Found most profitable product categories
* Analyzed seasonal sales trends



---

## 🔹 Conclusion

This project demonstrates SQL skills, ETL concepts, and data visualization techniques.
